## xml resource naming rules
- {widget}_{function}_{state}.xml
- id : camel
- color{colorName}{opacity} (camel case)
- dimens
  - snake case
  - {screen}_{margin / padding}_{direction}_{target1}_{target2}
- strings
  - snake case
  - {screen}_{txt / btn / hint}_{target}
